# Jiny CLI

Help to create boilerplate

## Install

```
$ curl -o /usr/local/bin/jiny https://raw.githubusercontent.com/tuhuynh27/jiny/master/cli/binaries/macos/jinycli
$ chmod +x /usr/local/bin/jiny
```

## Run

```
$ jiny
```
